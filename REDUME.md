# LHRPG Character Data Exporter (GAS)

ログ・ホライズンTRPG 公式サイトのキャラクターポータルからデータを取得し、外部ツール用のJSON形式を出力するGoogle Apps Scriptです。

## 機能
- 指定したキャラクターIDのステータス（HP、因果力、能力値など）を抽出
- ユドナリウム等の駒データとして読み込めるJSONフォーマットを生成

## 使い方
1. Google Apps Script プロジェクトを作成します。
2. `index.html` と `code.gs` を作成し、コードを貼り付けます。
3. ウェブアプリとしてデプロイします。

## 注意事項
- このスクリプトは個人利用を目的としています。
- 外部サイト（lhrpg.com）への負荷を考慮し、過度な連続実行は避けてください。
